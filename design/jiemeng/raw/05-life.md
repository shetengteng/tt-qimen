# 生活篇（life）

> 来源：公版《周公解梦》，部分条目参补《梦林玄解》。保留古籍原貌，未作改写。

### 梦见飞翔

**keywords**: 飞翔, 飞, 飞起来, 飞天, 翱翔
**source**: 《周公解梦》公版古籍
**tags**: auspicious

身飞举者，贵人助，禄位至；女人梦飞，主有喜。

### 梦见被追逐

**keywords**: 追逐, 被追, 追赶, 被人追, 被追赶
**source**: 《周公解梦》公版古籍
**tags**: cautious

见人追而不及者，主虚惊；被追入宅者，主客至。

### 梦见考试

**keywords**: 考试, 考场, 考卷, 忘带笔
**source**: 民间解梦常识
**tags**: cautious

梦中应试，主近有评选之事；忘带笔墨，心有所忧，事未备齐。

### 梦见结婚

**keywords**: 结婚, 婚礼, 嫁人, 娶亲, 成亲
**source**: 《周公解梦》公版古籍
**tags**: auspicious

见人嫁娶，主有喜庆；自成婚者，主迁改之事。

### 梦见迷路

**keywords**: 迷路, 迷失, 找不到路, 走丢
**source**: 民间解梦常识
**tags**: neutral

梦中迷途，主心事未定；路不得归，近有犹豫之事。

### 梦见坠落

**keywords**: 坠落, 掉下去, 从高处坠落, 跌落
**source**: 《周公解梦》公版古籍
**tags**: cautious

堕高而下，主失意；落井得出，主化险为夷。

### 梦见赛跑

**keywords**: 赛跑, 奔跑, 赛走, 跑步
**source**: 《周公解梦》公版古籍
**tags**: neutral

奔走如飞，主近日事多；赛跑得先，主争竞可胜。

### 梦见争吵

**keywords**: 争吵, 吵架, 口角, 相骂
**source**: 《周公解梦》公版古籍
**tags**: cautious

与人争辩，主近有口舌是非；梦吵而止，事多可解。

### 梦见打架

**keywords**: 打架, 斗殴, 打斗, 争斗
**source**: 《周公解梦》公版古籍
**tags**: cautious

与人相搏，主烦心忧事；打斗得胜，主事可平息。

### 梦见舞蹈

**keywords**: 舞蹈, 跳舞, 歌舞
**source**: 《周公解梦》公版古籍
**tags**: ambiguous

梦见歌舞，近有喜事；独自起舞，主心有所欲，未有共鸣。

### 梦见唱歌

**keywords**: 唱歌, 歌唱, 吟唱
**source**: 《周公解梦》公版古籍
**tags**: auspicious

高声吟唱，主心事舒畅；众人同歌，主和合之象。

### 梦见吃饭

**keywords**: 吃饭, 宴席, 赴宴, 饮食
**source**: 《周公解梦》公版古籍
**tags**: auspicious

与人共饭，主得人助；宴席丰美，主近有喜事。

### 梦见睡觉

**keywords**: 睡觉, 酣睡, 入眠
**source**: 《周公解梦》公版古籍
**tags**: neutral

梦中酣睡，主心神安稳；睡而惊醒，近有所忧。

### 梦见写字

**keywords**: 写字, 书写, 执笔
**source**: 《梦林玄解》
**tags**: auspicious

执笔而书，主学业有成；所书字迹端秀，主名声可起。

### 梦见读书

**keywords**: 读书, 看书, 诵读
**source**: 《周公解梦》公版古籍
**tags**: auspicious

读书明义，主学问大进；诵经而悟，近有师教。

### 梦见旅行

**keywords**: 旅行, 出行, 远行
**source**: 《周公解梦》公版古籍
**tags**: auspicious

出行远方，主有所得；归途顺遂，主事业可成。
