## 附录 D · 风险与对策（设计文档 §8）

| 风险 | 应对 |
|------|------|
| 月历 31 天 × 多字段渲染性能 | 当月数据按月缓存；切月时仅替换 31 个格子；避免每格内嵌大组件 |
| 跨农历月边界显示 | 日历格同时显示公历（大）+ 农历（小） |
| 用户"过度依赖"吉日 | 顶栏 disclaimer 始终可见；详情抽屉底部加"参考即可"提示 |
| `tyme4ts` API 升级 | 锁定 `^1.4.6`；core 内做容错 |
| 多语种事由名歧义（"安床" vs "moving in"） | 文档明确采用大陆通用版本术语；英文用音译+解释 |
