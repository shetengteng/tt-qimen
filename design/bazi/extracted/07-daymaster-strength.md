# 日主强弱文案 · 摘录

## 用途

本文件是 `src/modules/bazi/data/interpretTemplate.ts` 的
`PARAGRAPH_1_BY_DAYMASTER`（**10 干 × 5 旺衰 = 50 段**）的 raw 锚点对照档。

代码中 50 段目前以"五行意象 + 旺衰象意"为主体文案（见 T-2.2-D 抽检结论，
已标 `PARAGRAPH_1_BY_DAYMASTER_AUDIT_STATUS = 'non-classical-verified'`）。
本档的作用：

1. **定位**每组"日干 × 四季"在 `raw/qiongtong-baojian/full.md` 中的精确行号
2. **摘录**每组的关键句（1-2 句，不做白话改写）
3. **映射**每组 raw 锚点对应 code 侧的哪个旺衰 key

后续 T-2.2-D-v2 阶段可基于本档把 50 段文案从"编者撰写"回填为
`auditStatus: 'verified' + source: '《穷通宝鉴》· XXXX · L###'`。

- `[DT · L###]` = 滴天髓 · 天干论（`raw/ditian-sui/full.md`）
