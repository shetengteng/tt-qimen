- **进展（2026-04-24）中央特写 · 视觉打磨轮**：
  - 用户反馈两个具体问题：(1) 国风特写的签号"数字被挤到最后"；(2) 等级徽章的半透明背景
    在深色 overlay 上"看不清""上/中/下"文字

    DOM 拼接结果为 `"第  签 72"`，数字视觉上落在最右。已统一为 `prefix + value + suffix`
    三段式（中文"第 N 签"/英文"No. N"），数字始终居中，主题差异由 CSS 字号/颜色承担
  - **根因 2**：简约徽章 bg alpha=0.20 + 亮色文字（`#86efac`/`#fcd34d`/`#fca5a5`），
    在 `rgba(20,15,40,0.85)` scrim 上对比度约 3:1；国风徽章 bg alpha=0.28 + 同色文字，
    对比度 <3:1，完全不达 AAA 甚至 AA 门槛
  - **新视觉方案**：
    - 简约：bg → `rgba(133/22/127 ...,.78)` 深色背景 + 浅色文字（`#D1FADF`/`#FEF3C7`/`#FEE2E2`）+
      加粗 weight 600 + border alpha 0.65 + inset high-light box-shadow + 文字 shadow
    - 国风：bg alpha 0.28 → 0.82 全深化 + 文字色统一 `gf-rice` 米色 + 用 border 颜色
      表达三 tier（`gf-jade`/`gf-gold`/`gf-cinnabar`）+ 印章风 box-shadow + 文字投影
  - **Playwright 实测**（dev server）：
    - simple×mid  bg `rgba(133,77,14,.78)` color `#FEF3C7` → 对比度 ≈ **12.5:1**
    - guofeng×down bg `rgba(110,28,18,.82)` color `#F6ECD6` → 对比度 ≈ **11:1**
    - guofeng×up   bg `rgba(46,83,67,.82)` color `#F6ECD6` → 对比度 ≈ **10:1**（全部 > WCAG AAA 7:1）
    - 签号结构校验：`第` + `40` + `签` / `第` + `3` + `签`，数字始终居中、prefix/suffix 对称
  - 代码清理：`LotteryCenterpiece.vue` 去掉不再需要的 `useThemeStore` + `isGuofeng` 计算属性
    + `qianLabel` i18n 路径（模板层统一不再走 label）；LingqianTitle 仍保留 label 路径（两行
    布局与本特写不冲突）
  - 指标：3 文件 53/30 行；type-check 通过；生产 build 4.3 秒通过；无新增 linter 错误
