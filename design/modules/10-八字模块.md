## 1. 模块定位

八字是 `tt-qimen` 的**核心专业支柱**。
- **产品角色**：对标专业算命站，让用户看到完整的四柱 + 大运时间轴 + 流年标签；所有解读基于"模板化组合"的规则生成，避免 LLM 自由发挥的幻觉。
- **MVP 范围**：命盘 + 五行 + 十神 + 大运（10 年一运）+ 流年（当年干支）
- **后续扩展**：分面解读（事业/感情/财富/健康）、流月流日、神煞等

## 3. 数据结构

### 3.1 BaziChart（主要输出）

```typescript
interface BaziChart {
  // 四柱基本信息
  pillars: {
    year:  PillarInfo
    month: PillarInfo
    day:   PillarInfo  // 日主
    hour:  PillarInfo
  }

  // 五行分布（权重化）
  elements: {
    wood: number
    fire: number
    earth: number
    metal: number
    water: number
  }

  // 十神
  tenGods: {
    yearGan:  TenGodType
    monthGan: TenGodType
    hourGan:  TenGodType
  }

  // 日主旺衰与喜用神
  strength: '极旺' | '偏旺' | '中和' | '偏弱' | '极弱'
  favorableElements: Array<'木' | '火' | '土' | '金' | '水'>

  // 大运（MVP 做）
  decades: DecadeFortune[]
  startAge: {
    year: number              // 起运公历年
    months: number            // 起运年龄（整数年 + 月份，如 "3 岁 4 个月"）
  }

  // 流年（MVP 做，默认当年 + 后续 10 年）
  flowYears: FlowYear[]
  currentFlowYear: FlowYear   // 引用其中当前年份的条目
}
```

### 3.2 子结构

```typescript
interface PillarInfo {
  gan: string      // 天干（甲、乙、丙...）
  zhi: string      // 地支（子、丑、寅...）
  nayin: string    // 纳音（海中金、炉中火...）
  element: string  // 五行
}

type TenGodType =
  | '比肩' | '劫财' | '食神' | '伤官'
  | '正财' | '偏财' | '正官' | '七杀'
  | '正印' | '偏印' | '日主'

interface DecadeFortune {
  gan: string                       // 大运天干
  zhi: string                       // 大运地支
  element: string                   // 五行
  startAge: number                  // 起始年龄
  endAge: number                    // 结束年龄
  startYear: number                 // 起始公历年
  endYear: number                   // 结束公历年
  tenGod: TenGodType                // 相对日主的十神
  tendency: 'favorable' | 'neutral' | 'unfavorable'  // 吉/中/凶
  hint: string                      // 一句话模板
}

interface FlowYear {
  year: number                      // 公历年（如 2026）
  gan: string                       // 流年天干
  zhi: string                       // 流年地支
  element: string                   // 五行
  tenGod: TenGodType                // 相对日主的十神
  tendency: 'favorable' | 'neutral' | 'unfavorable'
  hint: string                      // 一句话模板
}
```

## 4. 核心算法

### 4.1 基础排盘（四柱 + 五行 + 十神）

```typescript
import { SolarTime } from 'tyme4ts'

function calculateBaziBasic(birth: BirthInfo): Partial<BaziChart> {
  const solar = SolarTime.fromYmdHms(
    birth.year, birth.month, birth.day,
    birth.hour, birth.minute, 0
  )
  const eightChar = solar.getLunarHour().getEightChar()

  const pillars = {
    year:  pillarFromSixtyCycle(eightChar.getYear()),
    month: pillarFromSixtyCycle(eightChar.getMonth()),
    day:   pillarFromSixtyCycle(eightChar.getDay()),
    hour:  pillarFromSixtyCycle(eightChar.getHour()),
  }

  const elements = calculateElements(eightChar)
  const tenGods  = calculateTenGods(eightChar, pillars.day.gan)
  const strength = judgeStrength(elements, pillars.day.gan)
  const favorableElements = deriveFavorable(strength, elements, pillars.day.gan)

  return { pillars, elements, tenGods, strength, favorableElements }
}
```

### 4.2 五行分布计算

```text
1. 天干：每个天干直接计五行，权重 1.0
2. 地支（按藏干权重）：
   - 主气（本气）×0.7
   - 中气 ×0.2
   - 余气 ×0.1
3. 求和到 5 个五行的总分
```

### 4.3 十神推算

以日主天干为基准，判断其他天干（年、月、时）与日主的关系：

| 关系 | 同五行 | 我生 | 我克 | 克我 | 生我 |
|------|-------|------|-----|-----|-----|
| 同阴阳 | 比肩 | 食神 | 偏财 | 七杀 | 偏印 |
| 异阴阳 | 劫财 | 伤官 | 正财 | 正官 | 正印 |

### 4.4 日主旺衰判断

```text
score = elements[日主五行]           // 日主本五行分
      + elements[生我的五行] × 0.5   // 印星生扶
      + 比肩劫财数 × 1              // 帮扶（月令得令加 ×2）

根据 score / total 比例分 5 档：
  > 50%  → 极旺
  40-50% → 偏旺
  30-40% → 中和
  20-30% → 偏弱
  < 20%  → 极弱
```

### 4.5 大运算法（MVP 做）

```typescript
import { Gender } from 'tyme4ts'

function calculateDecades(eightChar, dayGan, gender, favorableElements): DecadeFortune[] {
  const yun = eightChar.getYun(gender)  // tyme4ts 原生 API
  const rawDecades = yun.getDecadeFortunes(10)

  return rawDecades.map((raw, index) => {
    const gan = raw.getSixtyCycle().getHeavenStem().getName()
    const zhi = raw.getSixtyCycle().getEarthBranch().getName()

    const element = getElementOfGan(gan)
    const tenGod  = calcTenGod(dayGan, gan)
    const tendency = judgeTendency(element, favorableElements)
    const hint = lookupDecadeHint(tenGod, tendency)  // 查表得到模板

    return {
      gan, zhi, element,
      startAge: raw.getStartAge(),
      endAge: raw.getEndAge(),
      startYear: raw.getStartYear(),
      endYear: raw.getEndYear(),
      tenGod, tendency, hint,
    }
  })
}
```

### 4.6 流年算法（MVP 做）

```typescript
import { LunarYear } from 'tyme4ts'

function calculateFlowYears(dayGan, favorableElements, span = 10): FlowYear[] {
  const currentYear = new Date().getFullYear()
  const years: FlowYear[] = []

  for (let y = currentYear; y < currentYear + span; y++) {
    const sixty = LunarYear.fromYear(y).getSixtyCycle()
    const gan = sixty.getHeavenStem().getName()
    const zhi = sixty.getEarthBranch().getName()
    const element = getElementOfGan(gan)
    const tenGod = calcTenGod(dayGan, gan)
    const tendency = judgeTendency(element, favorableElements)
    const hint = lookupFlowYearHint(tenGod, tendency)

    years.push({ year: y, gan, zhi, element, tenGod, tendency, hint })
  }

  return years
}
```

### 4.7 吉凶判断（tendency）

```text
if element ∈ favorableElements     → 'favorable'
else if element 属于克日主的五行     → 'unfavorable'
else                                → 'neutral'
```

## 10. 风险与对策

| 风险 | 应对 |
|-----|------|
| 大运起运算法分歧（流派差异）| `tyme4ts` 支持配置，默认用通用算法 |
| 子时换日规则（早子/晚子） | 文档明确采用 `tyme4ts` 默认规则 |
| 真太阳时对不懂用户的困惑 | 默认关闭，高级选项 |
| LLM 生成的模板重复率高 | 每条十神 × 吉凶 写 3 种措辞变体，随机取 1 |
| 移动端大运时间轴显示挤 | 2 列 5 行蛇形网格，卡片竖向换行，避免横向滚动带来的视距损失 |
| 流年 10 年太多挤爆 | 默认只显示当年，展开按钮查看后续 |
