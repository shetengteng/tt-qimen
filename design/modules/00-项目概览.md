# 项目概览（tt-qimen）

> **所属文档**：方案设计 · 子模块  
> **版本**：v1.0（基于总体方案 v1.3）

## 1. 项目定位

**tt-qimen**（启门 · 启门问卜）是一个 **100% 静态**的传统占卜文化门户。

- **核心卖点**：8 合 1 占卜方法集合 + 专业级八字命盘（含大运 + 流年）
- **运行约束**：纯前端 · 无后端 · 无数据库 · 无鉴权 · 无付费
- **用户数据**：仅本地存储（localStorage），不上传
- **合规**：所有内容声明"仅供文化娱乐体验"

## 3. 技术栈

| 层 | 技术 | 说明 |
|---|------|------|
| 框架 | Vue 3 + `<script setup>` | Composition API |
| 构建 | Vite 5.x | 极速 HMR + 按模块代码分割 |
| 路由 | Vue Router 4 | 懒加载所有模块页 |
| 状态 | Pinia | 持久化 BirthInfo 到 localStorage |
| 工具 | VueUse | useStorage、useDark |
| 样式 | Sass + CSS 变量 | 支持主题切换 |
| **命理核心** | **`tyme4ts`** | 八字/大运/流年/黄历一站式 |
| 笔画 | `chinese-character-strokes` | 姓名学 |
| 搜索 | `fuse.js` | 解梦模糊匹配 |
