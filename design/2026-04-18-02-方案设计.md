## 待评审决策点（P0）

以下事项需要用户在启动开发前确认：

1. **[UI 风格二选一]** 国风 vs 现代简约（等待 HTML 原型产出后选择）
2. **[内容审校人员]** C 类 LLM 文案的审校负责人（至少 1 人）
3. **[部署平台]** Vercel / Cloudflare Pages / GitHub Pages 择一
4. **[域名]** 是否自备域名，或使用子域名
5. **[监控上报]** 是否引入 Sentry/Analytics（涉及隐私声明）
6. **[商业化]** 是否在 MVP 上线时即加入广告位预留
