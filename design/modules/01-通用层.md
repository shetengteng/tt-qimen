## 2. Pinia Store

### 2.1 `stores/user.ts`

```typescript
import { defineStore } from 'pinia'
import { useStorage } from '@vueuse/core'

export const useUserStore = defineStore('user', () => {
  const birth = useStorage<BirthInfo | null>('qimen.birth', null)
  const name  = useStorage<string | null>('qimen.name', null)

  function setBirth(b: BirthInfo) { birth.value = b }
  function setName(n: string)     { name.value = n }
  function clear() {
    birth.value = null
    name.value = null
  }

  return { birth, name, setBirth, setName, clear }
})
```

### 2.2 `stores/theme.ts`

```typescript
import { useDark, useToggle } from '@vueuse/core'

export const useThemeStore = defineStore('theme', () => {
  const isDark = useDark()
  const toggle = useToggle(isDark)
  return { isDark, toggle }
})
```

## 4. 共享组件

### 4.1 `AppHeader.vue`

- 桌面：LOGO + 3 主链接 + "更多 ▾" 下拉
- 移动：LOGO + 汉堡按钮 → 抽屉
- 支持 dark mode 切换（右上角图标按钮）

### 4.2 `DisclaimerBanner.vue`

- 始终可见的底部免责声明
- 内容："本站内容仅供文化娱乐体验，不构成任何预测或建议。"

### 4.3 `BirthInput.vue`

- 接受：`v-model:birth` / `@submit`
- 字段：出生日期、时间、性别、是否真太阳时（可选）
- 校验：1900-01-01 ~ 今天
- 复用：八字、称骨、大运计算

### 4.4 `ResultCard.vue`

- 通用结果卡片壳，提供：
  - `<slot name="header">` / `<slot name="body">` / `<slot name="footer">`
  - 统一的边框、阴影、间距
  - `level` prop 控制吉凶色条（上上/中吉/下下...）

### 4.5 `ShareModal.vue`

- 接受：`:cardData`
- 功能：渲染分享卡（含模块主结果 + 主题纹样 + 免责声明 + 二维码）→ 用 `html2canvas` 导出 PNG → 用户下载或复制到剪贴板

## 7. 全局错误处理

- 所有 `core/*.ts` 函数捕获异常，向外抛 `FortuneError`（含 code 和 userMessage）
- UI 层统一捕获，展示于错误态
- 非法日期、跨年边界问题均输出友好提示，不显示技术堆栈
