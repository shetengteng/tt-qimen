## 0. 模块定位

`tt-qimen` 的**核心专业支柱**之一（与紫微并列"命理双擎"）。
- 对标专业算命站，让用户看到完整的四柱 + 大运时间轴 + 流年标签。
- 所有解读基于"模板化组合"的规则生成，避免 LLM 自由发挥的幻觉。
- MVP 范围：命盘 + 五行 + 十神 + 大运（10 年一运）+ 流年（10 年）。

## 附录 D · 风险与对策（设计文档 §10）

| 风险 | 应对 |
|------|------|
| 大运起运算法分歧（流派差异） | 锁定 `tyme4ts` 默认；文档说明流派来源 |
| 子时换日（早子/晚子） | 跟 `tyme4ts` 默认规则；文档明确告知 |
| 真太阳时对小白用户的困惑 | 默认关闭，作为高级选项 |
| 流年模板维度从 30 → 300 段，工程量爆 | 分批 LLM 生成 + 抽样审校；保留旧模板做 fallback |
| 移动端大运时间轴显示挤 | 已用 2×5 蛇形网格，避免横向滚动 |
| 流年 10 年挤爆 | 默认仅展示当年，下拉展开 |
