## 附录 D · 风险与对策（设计文档 §8）

| 风险 | 应对 |
|------|------|
| 搜索性能（9000 条 × 多字段模糊） | fuse.js 索引化；> 3000 条时考虑 Web Worker；本地实测决定 |
| 数据集错漏（编码、重复） | 抽样校验 + 去重脚本 |
| 迷信引导风险 | 每条详情底部加"梦境解读仅为文化参考，不作行为指导" |
| 敏感词（死亡、宗教） | 简单替换词典过滤；谨慎显示 |
| `fuse.js` + 数据集首屏体积膨胀 | dynamic import；进入页面才加载（gzip ~60 KB JSON + ~10 KB fuse） |
| 多语种梦境词条 | 古文不译；只译 UI 框架与现代化建议 |
