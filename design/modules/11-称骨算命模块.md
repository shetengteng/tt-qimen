# 称骨算命模块

> **所属文档**：方案设计 · 子模块  
> **版本**：v1.0

## 1. 模块定位

- **方法**：袁天罡骨重歌诀 — 将年、月、日、时各对应一个骨重，相加查歌诀
- **可行性**：25/25 分 🟢 强推荐（见可行性分析 §3.1 方法 1）
- **复杂度**：极低（核心只需 4 个查表 + 1 次加法）

## 2. 数据结构

```typescript
type ChengguLevel = 'top' | 'high' | 'middle' | 'low'

interface ChengguPoem {
  weight: string        // "二两二"
  weightNumber: number  // 2.2
  poem: string[]        // 四句歌诀
  level: ChengguLevel   // 富贵等级
  description: string   // 现代白话解读（LLM 生成 + 审校）
}

interface ChengguResult {
  totalWeight: number   // 总骨重（两）
  displayWeight: string // "四两二"
  yearWeight: number
  monthWeight: number
  dayWeight: number
  hourWeight: number
  poem: ChengguPoem
}
```

## 3. 算法

```text
1. BirthInfo → 农历日期（tyme4ts：SolarDay.fromYmd(...).getLunarDay()）
2. 取年柱地支序号 → 查 YEAR_WEIGHT 表 → 年重
3. 取农历月份 → 查 MONTH_WEIGHT 表 → 月重
4. 取农历日 → 查 DAY_WEIGHT 表 → 日重
5. 取出生时辰（按时干支）→ 查 HOUR_WEIGHT 表 → 时重
6. 总骨重 = 年 + 月 + 日 + 时
7. 在 poems.ts 中查找对应骨重的歌诀
```

### 3.1 关键代码

```typescript
import { SolarDay } from 'tyme4ts'
import { YEAR_WEIGHT, MONTH_WEIGHT, DAY_WEIGHT, HOUR_WEIGHT } from '../data/chenggu/weights'
import { POEMS } from '../data/chenggu/poems'

export function calculateChenggu(birth: BirthInfo): ChengguResult {
  const solarDay = SolarDay.fromYmd(birth.year, birth.month, birth.day)
  const lunar = solarDay.getLunarDay()

  const yearGanZhi = lunar.getLunarMonth().getLunarYear().getSixtyCycle().getName()
  const lunarMonth = lunar.getLunarMonth().getMonth()
  const lunarDay = lunar.getDay()
  const hourIndex = Math.floor((birth.hour + 1) / 2) % 12

  const y = YEAR_WEIGHT[yearGanZhi]
  const m = MONTH_WEIGHT[lunarMonth]
  const d = DAY_WEIGHT[lunarDay]
  const h = HOUR_WEIGHT[hourIndex]

  const total = y + m + d + h
  const poem = POEMS.find(p => Math.abs(p.weightNumber - total) < 0.01)

  return {
    totalWeight: total,
    displayWeight: formatWeight(total),  // "四两二"
    yearWeight: y, monthWeight: m, dayWeight: d, hourWeight: h,
    poem: poem!,
  }
}
```

## 8. 风险

| 风险 | 应对 |
|-----|------|
| 农历与公历跨年的边界处理 | 用 `tyme4ts` 默认规则，子时换日按其配置 |
| 骨重小数精度（2.2 vs 2.20）| 用 `Math.abs(a - b) < 0.01` 比较 |
| 51 首歌诀中部分版本差异 | 文档注明采用版本（以"百度百科通用版"为准，白话为现代补充）|
