# 人物篇（people）

> 来源：公版《周公解梦》，部分条目参补《梦林玄解》。保留古籍原貌，未作改写。

### 梦见死人

**keywords**: 死人, 梦见死人, 尸体, 过世的人, 亡者
**source**: 《周公解梦》公版古籍
**tags**: ambiguous, auspicious

与死人言，主禄至；死人复生，主有吉事。

### 梦见已故亲人

**keywords**: 已故亲人, 过世亲人, 去世的家人, 祖父, 祖母, 亡亲
**source**: 《周公解梦》公版古籍
**tags**: neutral

见亡亲言语者，宜守本分，勿贪进取。

### 梦见陌生人

**keywords**: 陌生人, 不认识的人, 路人
**source**: 《周公解梦》公版古籍
**tags**: neutral

与不识之人相对，主遇新识；接陌路之言，近有贵人。

### 梦见小孩

**keywords**: 小孩, 孩子, 婴儿, 小儿
**source**: 《周公解梦》公版古籍
**tags**: ambiguous

抱儿者，主得喜事；婴儿啼哭，主病疾。

### 梦见怀孕

**keywords**: 怀孕, 怀胎, 有喜, 大肚子
**source**: 《周公解梦》公版古籍
**tags**: auspicious

妇人有身，主大吉；见孕妇入宅，主财至。

### 梦见生子

**keywords**: 生子, 生儿, 生女, 分娩
**source**: 《周公解梦》公版古籍
**tags**: auspicious

妇人生男，主贵；妇人生女，主富；产子而吉，家室兴。

### 梦见朋友

**keywords**: 朋友, 故人, 友人, 老友
**source**: 《周公解梦》公版古籍
**tags**: auspicious

故人相见，主有所遇；与友同坐，主得助力。

### 梦见亲人团聚

**keywords**: 团聚, 家人团圆, 合家欢, 团圆
**source**: 《周公解梦》公版古籍
**tags**: auspicious

合家欢聚，主家道兴隆；亲人相聚于堂，主吉事重至。

### 梦见父母

**keywords**: 父母, 父, 母, 双亲
**source**: 《周公解梦》公版古籍
**tags**: neutral

梦见父母，主添寿；父母俱安，主家和万事兴。

### 梦见妻子

**keywords**: 妻子, 夫妻, 妻, 内人
**source**: 《周公解梦》公版古籍
**tags**: neutral

妻来相就，主和好；妻与人言，防口舌。

### 梦见情人

**keywords**: 情人, 爱人, 恋人
**source**: 《周公解梦》公版古籍
**tags**: ambiguous

见恋人者，情思未断；与情人言欢，近有喜事。

### 梦见长辈

**keywords**: 长辈, 前辈, 尊长
**source**: 《梦林玄解》
**tags**: auspicious

得尊长教导，主有所进；与长辈对语，主得人扶。

### 梦见师长

**keywords**: 师长, 老师, 先生, 夫子
**source**: 《周公解梦》公版古籍
**tags**: auspicious

见师长而受教，主学业有成；师训相勉，近有贵人扶。

### 梦见官吏

**keywords**: 官吏, 官员, 大官, 做官
**source**: 《周公解梦》公版古籍
**tags**: ambiguous

见官者，主有口舌；得官者，主得禄；官吏入门，主讼事将临。

### 梦见旧识

**keywords**: 旧识, 老朋友, 同学, 旧友
**source**: 《周公解梦》公版古籍
**tags**: neutral

见旧识故人，主有所忆，心有未竟之事。

### 梦见圣贤

**keywords**: 圣贤, 圣人, 贤人
**source**: 《梦林玄解》
**tags**: auspicious

见圣贤者，主有大名；与圣人同坐，主得道之象。

### 梦见乞丐

**keywords**: 乞丐, 乞儿, 讨饭
**source**: 《周公解梦》公版古籍
**tags**: ambiguous

见乞丐者，主破财而得名；施乞者，积阴德之象。
