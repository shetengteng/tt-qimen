# 姓名学模块（五格剖象法）

> **所属文档**：方案设计 · 子模块  
> **版本**：v1.0

## 1. 模块定位

- **方法**：日本五格剖象（1918 熊崎健翁创立）— 将姓名按笔画拆为天格、人格、地格、外格、总格
- **可行性**：21/25 分 🟡 可做（见可行性分析 §3.2 方法 9）
- **体验定位**：实用工具型 — 用户测自己名字 / 起名参考

## 2. 数据结构

```typescript
interface NumerologyEntry {
  number: number                       // 1-81
  level: '大吉' | '吉' | '中吉' | '中平' | '凶' | '大凶'
  element: '木' | '火' | '土' | '金' | '水'
  summary: string                      // 一句话评价
  description: string                  // 详细解读（公版 + LLM 补充）
}

interface XingmingResult {
  fullName: string
  chars: Array<{
    char: string
    strokes: number                    // 康熙字典笔画
  }>

  // 五格
  grids: {
    tian: GridInfo                     // 天格
    ren:  GridInfo                     // 人格
    di:   GridInfo                     // 地格
    wai:  GridInfo                     // 外格
    zong: GridInfo                     // 总格
  }

  // 综合评分（每格按等级加权）
  overallScore: number                 // 0-100
  overallLevel: NumerologyEntry['level']
}

interface GridInfo {
  number: number
  entry: NumerologyEntry
}
```

## 3. 算法

### 3.1 笔画查询

使用 `chinese-character-strokes` 库：

```typescript
import charStrokes from 'chinese-character-strokes'

function getStrokes(char: string): number {
  return charStrokes[char] ?? throw new Error(`未收录字：${char}`)
}
```

### 3.2 五格公式

```text
输入姓名（按字拆分，如"王小明" → ['王', '小', '明']）

- 天格 = 姓笔画 + 1（单姓）/ 姓 1+ 姓 2（复姓）
- 人格 = 姓末字 + 名首字
- 地格 = 名首字 + 名次字（单名则名字 + 1）
- 总格 = 所有字笔画之和
- 外格 = 总格 - 人格 + 1

单名/复姓特殊处理：参照熊崎健翁流派
```

### 3.3 关键代码

```typescript
export function calculateXingming(fullName: string): XingmingResult {
  const chars = fullName.split('').map(c => ({ char: c, strokes: getStrokes(c) }))
  const isCompound = chars.length >= 3  // 2 字名、3 字名拆分
  const surnameLen = detectSurnameLength(fullName)  // 1 或 2

  // 各格计算
  const tian = computeTian(chars, surnameLen)
  const ren  = computeRen(chars, surnameLen)
  const di   = computeDi(chars, surnameLen)
  const zong = chars.reduce((sum, c) => sum + c.strokes, 0)
  const wai  = zong - ren + 1

  return {
    fullName,
    chars,
    grids: {
      tian: { number: tian, entry: NUMEROLOGY[tian] },
      ren:  { number: ren,  entry: NUMEROLOGY[ren]  },
      di:   { number: di,   entry: NUMEROLOGY[di]   },
      wai:  { number: wai,  entry: NUMEROLOGY[wai]  },
      zong: { number: zong, entry: NUMEROLOGY[zong] },
    },
    overallScore: calculateScore([tian, ren, di, wai, zong]),
    overallLevel: weightedLevel([tian, ren, di, wai, zong]),
  }
}
```

## 8. 风险

| 风险 | 应对 |
|-----|------|
| 多音字笔画差异 | 默认取该字最常用字形的笔画；提供高级用户手动修正入口 |
| 生僻字不在库 | `chinese-character-strokes` 覆盖 20000 字已足够 99% 场景 |
| 复姓识别（如"欧阳"）| 内置常见复姓列表：欧阳、司马、诸葛、上官等 |
| 用户期望"给宝宝起名" | 声明："本模块为测名参考，不提供起名建议" |
