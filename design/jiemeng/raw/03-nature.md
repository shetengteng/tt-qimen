# 自然篇（nature）

> 来源：公版《周公解梦》，部分条目参补《梦林玄解》。保留古籍原貌，未作改写。

### 梦见水

**keywords**: 水, 大水, 清水, 浊水, 洪水
**source**: 《周公解梦》公版古籍
**tags**: ambiguous

清水盈庭，主大吉；浑水满屋，主疾病。

### 梦见洪水

**keywords**: 洪水, 大水漫, 水淹, 水灾
**source**: 《周公解梦》公版古籍
**tags**: cautious

水漫宅舍，主忧疾；水退井盈，吉事至。

### 梦见火

**keywords**: 火, 大火, 着火, 火烧
**source**: 《周公解梦》公版古籍
**tags**: ambiguous

火光冲天，主大吉；火焚屋舍，主得财。

### 梦见太阳

**keywords**: 太阳, 日出, 朝阳, 日光
**source**: 《周公解梦》公版古籍
**tags**: auspicious

日初出，贵人至；日高升，名显扬；日落西，事将终。

### 梦见月亮

**keywords**: 月亮, 明月, 月圆, 月落
**source**: 《周公解梦》公版古籍
**tags**: auspicious

月圆光皎，百事吉；月蚀者，病疾至。

### 梦见打雷

**keywords**: 雷, 打雷, 响雷, 雷电
**source**: 《周公解梦》公版古籍
**tags**: neutral

雷鸣于野，行人利；雷击屋者，惊中有定。

### 梦见下雨

**keywords**: 雨, 下雨, 细雨, 暴雨, 久雨
**source**: 《周公解梦》公版古籍
**tags**: ambiguous

细雨润物，主有小利；暴雨骤来，防险事；久雨方晴，主忧事终结。

### 梦见下雪

**keywords**: 雪, 下雪, 瑞雪, 大雪
**source**: 《周公解梦》公版古籍
**tags**: auspicious

瑞雪降庭，主丰年有余；踏雪独行，主高洁之志。

### 梦见星辰

**source**: 《周公解梦》公版古籍

**source**: 《周公解梦》公版古籍

**source**: 《周公解梦》公版古籍

**source**: 《周公解梦》公版古籍
**tags**: auspicious

### 梦见大地

**source**: 《周公解梦》公版古籍
**tags**: neutral
