# 其他篇（other）

> 来源：公版《周公解梦》，部分条目参补《梦林玄解》。保留古籍原貌，未作改写。
> 本篇汇整不便归入前七类的物品 / 场景 / 符号象征。

### 梦见捡钱

**keywords**: 捡钱, 拾金, 钱, 钱币, 纸币
**source**: 《周公解梦》公版古籍
**tags**: auspicious

路上拾金，主进财；拾钱而散，主破财。

### 梦见镜子

**keywords**: 镜子, 照镜, 镜破, 铜镜
**source**: 《周公解梦》公版古籍
**tags**: ambiguous

镜光照人，主名显；镜破损，主夫妇分离。

### 梦见船

**keywords**: 船, 坐船, 大船, 航船
**source**: 《周公解梦》公版古籍
**tags**: ambiguous

乘船出海，主财至；船翻者，主阻滞。

### 梦见走路

**keywords**: 走路, 行走, 走在路上, 小路, 大路
**source**: 《周公解梦》公版古籍
**tags**: neutral

大道坦荡，百事顺遂；泥路难行，主阻滞。

### 梦见衣服

**keywords**: 衣服, 新衣, 旧衣, 穿衣
**source**: 《周公解梦》公版古籍
**tags**: auspicious

得新衣者，主喜事；衣履破损，主近有口舌。

### 梦见鞋

**keywords**: 鞋, 鞋子, 新鞋
**source**: 《周公解梦》公版古籍
**tags**: ambiguous

得新鞋，主有所行；鞋破裂，主近有远行之扰。

### 梦见书

**keywords**: 书, 书本, 书籍, 典籍
**source**: 《周公解梦》公版古籍
**tags**: auspicious

得书者，主学问有进；阅古书，主有所悟。

### 梦见笔

**keywords**: 笔, 毛笔, 钢笔
**source**: 《梦林玄解》
**tags**: auspicious

得笔者，主有文名；笔未能书，主心有未成之文。

### 梦见宝物

**keywords**: 宝物, 珠宝, 金银, 珍珠
**source**: 《周公解梦》公版古籍
**tags**: auspicious

得宝者，主有大财；宝物失散，主近有损耗。

### 梦见花朵

**keywords**: 花朵, 鲜花, 百花, 花开
**source**: 《周公解梦》公版古籍
**tags**: auspicious

百花齐放，主家庭和睦；花开见果，主事业有成。

### 梦见树木

**keywords**: 树木, 大树, 老树
**source**: 《周公解梦》公版古籍
**tags**: auspicious

大树参天，主家运兴旺；枯树发新枝，主困境转机。

### 梦见果实

**keywords**: 果实, 果子, 水果, 桃李
**source**: 《周公解梦》公版古籍
**tags**: auspicious

得果而食，主心愿可遂；果实丰收，主家业有成。

### 梦见礼物

**keywords**: 礼物, 受赠, 赠物
**source**: 《周公解梦》公版古籍
**tags**: auspicious

受人赠物，主近得贵人扶助；赠物与人，主阴德之积。

### 梦见信件

**keywords**: 信件, 书信, 家信
**source**: 《周公解梦》公版古籍
**tags**: neutral

得家书者，主远人有讯；书信破损，近有讯息之扰。

### 梦见钥匙

**keywords**: 钥匙, 钥
**source**: 《梦林玄解》
**tags**: auspicious

得钥开门，主能解困；失钥寻之，心有未通之事。

### 梦见笑声

**keywords**: 笑, 笑声, 欢笑
**source**: 《周公解梦》公版古籍
**tags**: ambiguous

众人欢笑，主家和事顺；独自而笑，反主心事暗生。

### 梦见哭泣

**keywords**: 哭, 哭泣, 流泪
**source**: 《周公解梦》公版古籍
**tags**: ambiguous

梦中大哭，主心事得以纾解；为死者哭，近有念旧之情。
