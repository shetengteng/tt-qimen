# 动物篇（animal）

> 来源：公版《周公解梦》，部分条目参补《梦林玄解》。保留古籍原貌，未作改写。

### 梦见蛇咬自己

**keywords**: 蛇, 梦见蛇, 蛇咬, 蛇咬我, 被蛇咬, 蛇咬自己
**source**: 《周公解梦》公版古籍
**tags**: ambiguous, auspicious, cautious

蛇咬人，主得财。蛇入怀，生贵子。蛇绕腰，主有孕。

### 梦见蛇吞物

**keywords**: 蛇吞物, 蛇, 蛇吞, 大蛇
**source**: 《周公解梦》公版古籍
**tags**: cautious

蛇吞物者，主异变，远行者忌之。

### 梦见白蛇

**keywords**: 白蛇, 蛇, 白色的蛇, 白蟒
**source**: 《周公解梦》公版古籍
**tags**: auspicious

白蛇入室，主添人口；白蛇盘门，吉兆临之。

### 梦见蛇化龙

**keywords**: 蛇化龙, 蛇变龙, 蛇成龙
**source**: 《梦林玄解》
**tags**: auspicious

蛇化为龙，主得贵人提携，上人见喜。

### 梦见龙

**keywords**: 龙, 飞龙, 见龙, 龙飞, 龙腾
**source**: 《周公解梦》公版古籍
**tags**: auspicious

龙飞者，贵人扶助之象；乘龙上天，必得显贵。

### 梦见龙入井

**keywords**: 龙入井, 龙在井, 井中龙
**source**: 《周公解梦》公版古籍
**tags**: cautious

龙入井者，贵人被囚之象，不吉。

### 梦见老虎

**keywords**: 虎, 老虎, 猛虎, 虎来, 白虎
**source**: 《周公解梦》公版古籍
**tags**: ambiguous, cautious

虎入宅，官禄至；虎伤人，主疾病；骑虎行，贵人助。

### 梦见虎走

**keywords**: 虎走, 老虎离去, 虎远去
**source**: 《周公解梦》公版古籍
**tags**: auspicious

虎出山走，主贵客至，客至有喜。

### 梦见鱼

**keywords**: 鱼, 活鱼, 大鱼, 捕鱼, 钓鱼, 鱼跃
**source**: 《周公解梦》公版古籍
**tags**: auspicious

鱼跃池中，财禄至；得大鱼，主厚利；食鱼肉，大吉。

### 梦见鱼游

**keywords**: 鱼游, 鱼在水, 鱼戏水
**source**: 《周公解梦》公版古籍
**tags**: auspicious

鱼游深水，主远行有利；鱼戏清波，主喜事临。

### 梦见马

**keywords**: 马, 骑马, 白马, 奔马
**source**: 《周公解梦》公版古籍
**tags**: auspicious

骑马奔驰，主喜事临门；见白马至，主有远客。

### 梦见牛

**keywords**: 牛, 水牛, 牛耕田, 耕牛
**source**: 《周公解梦》公版古籍
**tags**: auspicious

牛耕田园，主事成丰收；乘牛入市，主有财至。

### 梦见狗

**keywords**: 狗, 犬, 犬吠, 狗咬
**source**: 《周公解梦》公版古籍
**tags**: ambiguous

犬吠主有客至；群犬相斗，防口舌是非；犬咬人，主有小病。

### 梦见鸟

**keywords**: 鸟, 飞鸟, 众鸟, 百鸟
**source**: 《周公解梦》公版古籍
**tags**: auspicious

百鸟飞集，主喜事重重；飞鸟入怀，主得贵子。

### 梦见凤凰

**keywords**: 凤凰, 凤, 鸾, 凤鸣
**source**: 《周公解梦》公版古籍
**tags**: auspicious

凤凰入怀，主生贵子；凤鸣于野，主名显远闻。

### 梦见鸡

**keywords**: 鸡, 公鸡, 母鸡, 鸡鸣
**source**: 《周公解梦》公版古籍
**tags**: auspicious

鸡鸣于晓，主官事顺遂；群鸡相逐，主和合之象。

### 梦见燕子

**keywords**: 燕, 燕子, 飞燕, 乳燕
**source**: 《周公解梦》公版古籍
**tags**: auspicious

燕入家室，主添丁进口；乳燕绕梁，主家庭和睦。

### 梦见鹤

**keywords**: 鹤, 仙鹤, 白鹤
**source**: 《周公解梦》公版古籍
**tags**: auspicious

鹤立庭中，主长寿延年；白鹤冲天，主名声显达。
