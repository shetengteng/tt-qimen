# tt-qimen

静态问卜站（Static Divination Web）—— 八个核心占卜模块的轻量化实现。

## 核心模块

- **八字命盘**：天干地支 / 五行 / 十神 / 简化大运时间轴
- **紫微斗数**：十二宫 / 四化飞星 / 三方四正 / 大限·小限
- **称骨算命 / 小六壬 / 灵签 / 姓名学 / 择日 / 解梦**

## 目录结构

```
design/                 设计文档与原型
├── prototypes/
│   ├── guofeng/        国风风格 HTML 原型
│   └── minimal/        现代简约风格 HTML 原型
└── docs/               方案设计、原型说明拆分文档
```

## 风格

每个模块提供两套 HTML 静态原型：`国风（guofeng）` 与 `现代简约（minimal）`，
采用原生 HTML/CSS/JS，无构建步骤，可直接在浏览器中打开。
