## 12. 风险与对策

| 风险 | 应对 |
|---|---|
| 搜索性能（9000 条 × 多字段模糊） | fuse.js 索引化；超过 3000 条时考虑 Web Worker |
| 数据集错漏（编码、重复） | 抽样校验 + 去重脚本 + zod schema 校验 |
| 迷信引导风险 | 每条都显示："梦境解读仅为文化参考，不作为行为指导" |
| 敏感词（死亡、宗教） | 简单替换词典过滤；谨慎显示，附"心理学常识"短语 |
| 多语言下中文词条难以英文搜索 | 英文版可显式提示"本工具基于中文词条，英文搜索可能命中率较低" |
