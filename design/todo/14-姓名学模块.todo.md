## 0. 模块定位

日本五格剖象法（1918 熊崎健翁创立）— 将姓名按笔画拆为天/人/地/外/总五格。
- 可行性 21/25 分 🟡 可做。
- 体验定位：**实用工具型** — 用户测自己名字 / 起名参考。
- 当前 `src/modules/xingming/XingmingPage.vue` 仅渲染 `ModulePlaceholder`，**算法/数据/UI 全 0**。

## 附录 D · 风险与对策（设计文档 §8）

| 风险 | 应对 |
|------|------|
| 多音字笔画差异 | 默认取最常用字形；高级允许手动修正 |
| 生僻字未收录 | 友好提示"暂未收录该字"，不阻塞其他字 |
| 复姓识别误判 | 内置白名单优先；模糊时让用户选 |
| 用户期望"给宝宝起名" | 文案声明"仅为测名参考，不提供起名建议" |
| `chinese-character-strokes` 包体积 | 必须走 dynamic import，仅在用户提交后加载 |
| 多语种笔画概念差异（英文/繁体） | 模块只对汉字生效；非汉字提示"仅支持中文姓名" |
