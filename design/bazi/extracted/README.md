# 八字文案 · 分类摘录

## 目录说明

本目录对 `design/bazi/raw/` 下的 7 部古籍进行**功能导向的二次加工**——

- `raw/`：原文全貌，适合通读、研究、核对
- `extracted/`（本目录）：按项目子功能**剪裁 + 定位 + 标注** 的工作版，适合白话改写时直接取用

## 抽取规则

每条摘录遵循以下模板：

```markdown
## {子项名，如"甲子乙丑海中金"}

**源**：{书名}·{卷/章} · 行号 {X-Y}
**原文**：
> {……古籍原文，保持竖排式句读，不改用字……}

**白话要点**：
- 要点 1
- 要点 2

**文案草稿（200 字以内）**：
{可直接写入 TypeScript 模板的白话版}

---
```

## 复用原则

1. **版权**：所有原文均为公版（作者去世超过 100 年），可自由使用
2. **署名**：在最终代码文件头部注明："文本改写自《{书名}·{卷}》维基文库/ctext 版 2026-04-20 采录"
3. **审校**：改写后必需人眼二校——古文直译白话易出语病
4. **字数**：短版 ≤20 字、长版 80-120 字；格局长解读 200 字（±20）
