## 2. 核心算法（`src/modules/xingming/core/xingming.ts` · **待新建**）

### 2.2 复姓识别

- [ ] `detectSurnameLength(fullName: string): 1 | 2`
- [ ] 内置常见复姓列表：欧阳 / 司马 / 诸葛 / 上官 / 司徒 / 慕容 / 端木 / 公孙 / 长孙 / 宇文 等

### 2.3 五格公式（熊崎健翁流派）

- [ ] `computeTian(chars, surnameLen)` — 单姓：姓 + 1 / 复姓：姓 1 + 姓 2
- [ ] `computeRen(chars, surnameLen)` — 姓末字 + 名首字
- [ ] `computeDi(chars, surnameLen)` — 名首字 + 名次字（单名则名字 + 1）
- [ ] `computeZong(chars)` — 所有字笔画之和
- [ ] `computeWai(zong, ren)` — 总格 - 人格 + 1
- [ ] `calculateXingming(fullName: string): XingmingResult` 主入口
- [ ] `calculateScore(grids: number[]): number` — 五格按等级加权评分（0-100）
- [ ] `weightedLevel(grids: number[]): XingmingLevel` — 加权等级

### 2.4 边界处理

- [ ] 字符串长度校验：`fullName.length ∈ [2, 5]`
- [ ] 输入含非汉字（数字 / 字母）→ FortuneError
- [ ] 多音字 → 取常用字形笔画（先 MVP，后续提供手动修正入口）

## 16. 风险与对策

| 风险 | 应对 |
|---|---|
| 多音字笔画差异 | 默认取常用字形；MVP 不提供手动修正，后续迭代 |
| 生僻字不在库 | `chinese-character-strokes` 覆盖 20000 字已足够 99% 场景；缺字时友好提示 |
| 复姓识别（如"欧阳"） | 内置 ~20 个常见复姓表；不在表中按单姓处理 |
| 用户期望"给宝宝起名" | 显著声明："本模块为测名参考，不提供起名建议" |
| `chinese-character-strokes` 包体积大（~200KB） | dynamic import，仅在用户提交时加载 |
