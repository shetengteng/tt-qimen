## 4.9 紫微斗数页 `ZiweiView.vue`（v1.4 新增 · 命理双擎之二）

#### 4.9.1 页面定位

紫微斗数模块对标八字，视觉上与八字采用"同源语言"（生辰输入 → 骨架屏 → 命盘 → 论命 → 大限 → 流年），但核心视图从八字的"四柱 + 五行雷达"替换为**紫微最具辨识度的 4×4 十二宫盘**。

#### 4.9.4 紫微页交互细节

| 元素 | 行为 |
|------|------|
| 12 宫盘 | 桌面 4×4 CSS Grid；移动默认展示主要 4 宫（命/财/官/夫）+ 其余 8 宫折叠为列表 |
| 宫位卡 | 悬停桌面端显示浅金描边；点击展开该宫的主星 + 辅星 + 杂耀完整列表 |
| 主星亮度 badge | 庙=金、旺=橘、得=青、利=青、平=灰、不=灰、陷=红；同八字吉凶编码哲学 |
| 身宫标记 | 宫位右下角显示 "★身宫" 图标 |
| 当前大限标记 | 宫位右下角显示 "当" 圆章（同八字大运"当"字样） |
| 命宫主星论命 | 120-180 字，关键词显示为芯片（同解梦条目的"tag"风格）|
| 四化卡 | 4 张横排卡片；桌面端 hover 展开 30 字详细解读 |
| 大限时间轴 | **与八字大运完全一致**（蛇形网格 + 虚线连接），桌面 6×2，移动 2×6 |
| 流派切换 ▾ | 默认三合派 + 中州派；Post-MVP 提供飞星派选项 |
| 再排一次 | 滚动到顶部并复用 BirthInput（同八字）|
| 错误态 | 未来日期、非法日期时，按钮 disabled + 红色提示 |
