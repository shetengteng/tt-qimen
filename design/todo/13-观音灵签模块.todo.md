**实际工程链路**（2026-04-24 落定）：
1. 抓取：`scripts/lingqian-data/{4 个 uuid}.txt`（搜狐 1-25 / 26-50 / 51-75 / 77-100 各分卷原文）
2. 规范化：`scripts/build-lingqian-dataset.mjs`
   - `splitSigns` + `extractSection` 抽 8 个原始字段（poem / 解曰 / 诗意 / 仙机 / 典故 等）
   - `humanizeOne` + `joinHumanized` + `spreadFreeform` 把仙机 15 项 → 6 大 topics 白话化
   - `smoothJieyue` 处理 4 字断句、`cleanScrapeTail` 剔除搜狐脚注、`cleanPunct` 修补拼接标点
   - `LINGQIAN_OVERRIDES`（`lingqian-overrides.mjs`）针对 5 签共用文案问题手工修正
   - `lingqian-76-100.mjs` 内嵌 76-100 公版数据
3. 校验：`scripts/validate-lingqian-dataset.mjs`（8 项检查 · 当前 0 errors / 0 warnings）
4. 输出：`src/modules/lingqian/data/guanyin.json`，运行时 `core/lingqian.ts::loadGuanyinData` 懒加载
5. Fallback：`data/items.ts` 6 签样本与 guanyin.json 同 id 数据**逐字段一致**（由 validate 强制）
