## 与 `extracted/` 的关系

- `raw/` 保留**原文格式**（包括序、跋、卷标题），仅做小幅清理（去掉网页导航、版权 footer、JS 元素）
- `extracted/` 按**线索**分类（纳音 / 格局 / 大运 / 流年 / 神煞），从 raw/ 中摘录关键段落，附带：
  1. 原文（繁体/简体保留）
  2. 简体翻译（如需）
  3. 现代解读（用于 i18n JSON 生成时的语料参考）
  4. 引用来源（书名 + 卷次 + 段落定位）

## 抓取方法

所有原文通过 `WebFetch` 工具获取（基于 Cursor / Claude 内置工具），转换为 markdown 格式后落盘。如需更新：

1. 重新抓取：调用 `WebFetch(url)` → 返回 markdown
2. 添加 YAML 头：`title / source / fetched_at / license`
3. 文件命名：`<book-pinyin>/full.md` 或 `<book-pinyin>/volume-NN.md`
4. 更新本 README 的 fetch 时间
